## 🤝 Contributing to Roadmap

### Como Sugerir Features

1. **GitHub Issues:** Abra uma issue com label `feature-request`
2. **Discussions:** Participe das discussões em GitHub Discussions
3. **Feedback:** Envie feedback via email ou formulário

### Como Votar em Features

- 👍 Reaja com thumbs up em issues que você quer ver
- 💬 Comente com casos de uso
- 🔗 Link para exemplos relacionados

### Priorização

Features são priorizadas baseado em:

1. **Impact:** Quantos usuários beneficiam?
2. **Effort:** Quanto trabalho é necessário?
3. **Community:** Quantas pessoas votaram?
4. **Strategic Fit:** Alinha com visão do projeto?

- Discoveries técnicas
- Feedback de usuários
- Mudanças prioritárias

### Como posso influenciar o roadmap?

1. Vote em features existentes
2. Abra novas issues com casos de uso
3. Participe das discussões
